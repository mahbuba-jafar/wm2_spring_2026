# WM2 Course – Class Notes

This repository hosts weekly class notes for the **Web and Mobile 2 course**.  
Each CRN has its own dedicated branch to keep the materials organized.

## Branches

- **`20328`** → Contains notes for CRN **20328**
- **`20329`** → Contains notes for CRN **20329**
- **`20924`** → Contains notes for CRN **20924**
- **`lab`** → Contains notes for **lab** sessions

## How to Access Notes

1. Select the branch that matches your CRN from the branch dropdown at the top of the repository.
2. Browse through the  folders/files based on modules to view the class notes.

## Contributing

If you notice mistakes or want to add resources:
1. Clone the repo and checkout the branch for your CRN.
2. Create a new branch off of the selected branch.
3. Commit your changes.
4. Open a Pull Request to the respective branch.
